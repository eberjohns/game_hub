# Configuration

Install necesssary packages:
```cmd
npm install
```

Run the app:
```cmd
node server.js
```

Client front side will be running at:
http://localhost:3000/

Access admin page at:
http://localhost:3000/admin.html


## How to use this?

- Access admin page
- Paste your game url, Generate PIN
- Access the client side website & enter the PIN, username
- When you start the game in Admin, all the players will have the game loaded immediately
- Njoy!
